OpentheClaw. One‑click SSH tunnel + Web login.

Your app's Web UI only listens on the server (for example http://127.0.0.1:18789/). You SSH in anyway—so why type the same commands every day? Configure it once in a browser and launch tunnel + browser with a single double‑click.

View on GitHub Read full README
Supported: macOS Linux Windows
You never have to
  • Open a terminal just to start a tunnel.
  • Type or remember long SSH commands.
  • Paste passwords into random terminals.
  • Switch tabs after connecting—browser opens for you.
How it works
  • Configure once in a local web page.
    Tell it how you SSH and which Web UI URL to open.
  • Use the OS‑specific launcher.
    A tiny script runs the Python helper and starts the tunnel.
  • Browser opens automatically.
    The helper warms the page so the first paint feels fast.
macOS
1. Open Finder.
2. Double‑click mac/Click-default-OpenClaw.command.
3. Your browser opens the configured Web UI.
Linux
1. In a terminal: linux/OpenClaw-linux.sh.
2. Or double‑click it in your file manager and choose “Run”.
3. The SSH tunnel starts and your Web UI opens.
Windows
1. Open Explorer.
2. Double‑click windows/OpenClaw-windows.bat.
3. Use the same launcher every day for the same Web UI.
